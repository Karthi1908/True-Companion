# True companion

An AI-powered conversational mental wellness agent application.

## Project Structure

- `backend/`: FastAPI server using Google ADK for agent orchestration.
- `frontend/`: Next.js web application for the user interface.

## Tech Stack

- **Backend**: Python, FastAPI, Google ADK, Gemini, Comet Opik.
- **Frontend**: React, Next.js, Tailwind CSS.
